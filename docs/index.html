原版，兼容

为了保证完全的移植，以及强大的兼容，RadiumOS所有功能皆在原版代码上进行开发，未使用任何非官方扩展。为了得到相对完整的功能，我几乎要掏空Scratch的内容了。不过仍旧有些功能是原版无法办到的……

简单，易懂

简单基础的代码，以及易懂的变量，列表和函数命名，简直不要太简单！不过，值得一提的是，没有任何注释（旧架构代码）！一切都要靠你自己理解！
也许重构的代码我会加上注释……
